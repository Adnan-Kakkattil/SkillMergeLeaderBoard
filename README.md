# Skill Merge Leaderboard

A live, animated CTF-style leaderboard built with React, Vite, Tailwind CSS, Framer Motion, and Lucide icons.

## Run Locally

```bash
npm install
npm run dev
```

Open the local URL shown in the terminal (usually `http://localhost:5173`).

## Production Build

```bash
npm run build
npm run preview
```

## Tech Stack

- React 18
- Vite 5
- Tailwind CSS 3
- Framer Motion
- Lucide React
